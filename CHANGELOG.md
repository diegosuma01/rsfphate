# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project follows semantic versioning in a lightweight way.

## [0.1.0] - 2026-04-16

### Added

- Initial public release of the `rsfphate` package.
- Public `RSFPhate` estimator API for end-to-end survival clustering.
- Survival-target helper `to_survival_array(...)`.
- Synthetic donut survival dataset generator for examples and smoke tests.
- Minimal example script under `examples/basic_usage.py`.
- Smoke-test coverage for package import and end-to-end fitting.
- Packaging metadata via `pyproject.toml`.
- `CITATION.cff` metadata for repository citation.
- `PUBLISHING.md` instructions for GitHub and optional PyPI release.
